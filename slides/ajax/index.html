AJAX

Work in progress

Asynchronous JavaScript And XML 

AJAX es una forma de desarrollo del uso de herramientas web
que proporciona una experiencia mucho más dinámica al usuario, de una manera que nunca antes se había visto en Internet.

  • Estándares de presentación: XHMTL y CSS
  • Interacción: DOM (Document Object Model)
  • Intercambio de datos: XML y XSLT
  • Interacción asíncrona con el servidor: XMLHttpRequest
  • Programación en el cliente: Javascript

Aplicaciones web

  • más interactiva
  • rápidas
  • atractivas

El navegador contiene una aplicación,
no un documento

El servidor no envía contenidos,
envia datos

El usuario interactúa de manera fluida y continua
muchas de las peticiones
son implícitas

Se trata de auténtico desarrollo de software

Ventajas

  • Interfaces de usuario más “ricas”
  • Menor consumo de ancho de banda
  • Intercambio de datos = servicios web, RSS, etc
  • Gran variedad de bibliotecas para facilitar el desarrollo

Objeto XMLHttpRequest

permite que el navegador recupere
datos (XML, texto plano, …) desde el servidor
en segundo plano
