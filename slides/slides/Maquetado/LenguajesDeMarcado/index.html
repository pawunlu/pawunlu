Standard Generalized Markup Language (SGML)

1986 -> International Standards Organization (ISO-8879)

Define la gramática de un lenguaje

Declarativo

Riguroso

SGML

NO es un lenguaje de marcas en si mismo

Meta-lenguaje

Para crear lenguajes de marcado para
el intercambio y conservación de documentos.

Un documento SGML

  • la Declaración SGML
  • el Prólogo DOCTYPE -> Definición de tipo de documento (DTD)
  • la instancia, que contiene un elemento superior y su contenido.

SGML

Simples -> MediaWiki

Entre corchetes -> RTF

Etiquetas coincidentes -> HTML
